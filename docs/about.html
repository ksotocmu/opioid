Project Background

This site was developed by Nicholas Carolan as part of the altREU program at Portland State University: Computational Models Serving Communities. This program is an eight-week research experience for undergraduates (REU) focused on building computational models for a specific need in the local community.

This project is not formally affiliated with the Pennsylvania Department of Health or any other government agency in any capacity.

Special thanks are in order to Dr. Wayne Wakeland, the advisor for the PA Opioid Prediction project, as well as Prof. Christof Teuscher, the program director, and MacKenzie Gray.

If you have additional data or resources you would like to contribute to this project, please contact Nicholas.
