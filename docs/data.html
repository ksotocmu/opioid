Data Sources

CDC WONDER Mortality Data

The Centers for Disease Control and Prevention maintains a database of underlying and multiple causes for deaths in the United States. Data is provided in deaths per 100,000 by county. For this project, data was queried for Pennsylvania using MCD - ICD-10 Codes: T40.0 (Opium); T40.1 (Heroin); T40.2 (Other opioids); T40.3 (Methadone); T40.4 (Other synthetic narcotics); T40.6 (Other and unspecified narcotics).

State Police Overdose Information Network Data (2018 - Current)

This dataset contains summary information on overdose responses and naloxone administrations by Pennsylvania criminal justice agencies and some third-party (i.e. EMS, Fire, Medical Staff, etc) first responders voluntarily entering incident data. Due to the voluntary nature of the application, the ODIN information provided may not represent the totality of all overdose and/or naloxone administration incidents involving criminal justice agencies occurring within the Commonwealth. Although this dataset does include some third-party administrations of naloxone, it should not be used to measure overdose response and naloxone administration incidents among all first responders.

Emergency Medical Services (EMS) Naloxone Doses Administered (2018 - Current)

This dataset contains information on the number of doses given for Naloxone by EMS throughout the Commonwealth of Pennsylvania by County. Data is updated Quarterly starting with the first Quarter of Calendar Year 2018.

Methodology

This project uses machine learning models to predict the likelihood of opioid overdoses and deaths across Pennsylvania. Machine learning models simply learn relationships from the training data they are provided. In this project, models are trained on data relating to opioid overdoses provided by the state and national government.

Building a high-quality machine learning model demands high-quality and high-volume data.

Model Architecture

Due to the different characteristics of the available data, this project elected to develop several models to predict different opioid-related statistics. The first approach, a regression model, tries to predict how many people (per 100,000) will die of an opioid overdose per year in a given county. This involves learning a relationship between other characteristics of that county (such as income, poverty, unemployment rates, etc.) and the level of opioid overdoses.

County-Level Regression

Prediction: Annual Deaths per 100,000 due to Opioids

Given certain statistics about a county, can we predict how many of its residents will die of opioid overdoses? This model analyzes demographic data to learn to predict each county's annual opioid death rate. Data on the rate of opioid overdose deaths is taken from the CDC. Demographic data consists of mean household income, unemployment rate, and individual poverty rate, and labor force participation rate, and is queried from the U.S. Census Bureau's American Community Survey.

The model is a simple feed-forward dense neural network for regression. It has two hidden layers, each with 64 neurons, and produces a single output corresponding to a county's opioid-induced death rate. The model is trained with RMSProp with learning rate = 0.001 to minimize mean squared error. It trains with early stopping based on validation set loss.

After the model is trained on its training data, it is evaluated on test data it has not seen before to assess its accuracy. This regression model explained 45.41% of the variance in the test data.

Time Series Forecasting

Prediction: Daily Naloxone Administrations

Every day, first responders administer Naloxone (Narcan®) to people undergoing an opioid overdose. This model takes in data from the tens of thousands of Naloxone administrations performed in PA over the last three years. Using a sophisticated deep learning model, it identifies trends and recurrences to predict the number of future overdose incidents.
